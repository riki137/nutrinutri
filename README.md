# <img src="assets/images/nutrinutri.svg" width="48" height="48" vertical-align="middle"> NutriNutri

> **Your Personal AI Nutritionist. Simple. Private. Open Source. Not-for-profit.**

**NutriNutri** is a cutting-edge **AI Nutrition Tracker** built with Flutter. It's designed to make calorie counting and macro tracking effortless and accurate. Forget scrolling through endless databases—just snap a photo or type what you ate, and let our AI do the rest.

I believe that health tools should be accessible to everyone and respect your privacy. That's why NutriNutri is completely **Non-Profit**, **Open Source (GPLv3)**, and designed with **Privacy First** principles.

## ✨ Key Features

- **📸 AI-Powered Auto-Logging**: Simply take a picture of your meal or describe it in plain English. Our advanced AI breaks down ingredients, estimates portions, and logs calories/macros instantly.
- **🥗 Smart Nutrition Analysis**: Get detailed breakdowns of Calories, Protein, Carbs, and Fats.
- **🔒 Privacy Focused**: Your health data belongs to you. No tracking, no selling data.
- **📱 Multi-Platform**: Seamless experience on Web, Android, Linux, macOS and Windows.
- **💸 Forever Free & Open**: Use your own OpenRouter API key to power the app.
- **Non-Profit**: I believe that health tools should be accessible to everyone and respect your privacy. That's why NutriNutri is completely **Non-Profit**, **Open Source (GPLv3)**, and designed with **Privacy First** principles.

## 🏁 Getting Started

### Prerequisites

- [Flutter SDK](https://flutter.dev/docs/get-started/install)
- [OpenRouter API Key](https://openrouter.ai/) (for AI features)

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/riki137/nutrinutri.git
   cd nutrinutri
   ```

2. **Install dependencies**

   ```bash
   flutter pub get
   ```

3. **Run the app**
   ```bash
   flutter run
   ```

### Configuration

1. Open the app.
2. Go to **Settings**.
3. Enter your **OpenRouter API Key**.
